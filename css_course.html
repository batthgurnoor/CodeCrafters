CSS Introduction

❮ Previous Next ❯

CSS stands for Cascading Style Sheet. It is widely used to design HTML and XML pages.

What is CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is a style sheet language used to describe the look and formatting of a document written in a markup language such as HTML or XML.
  • CSS allows web developers to separate the presentation of a web page from its content
  • CSS works by defining rules that apply to specific elements on a web page.
  • CSS can be applied to a web page in several ways, including as an external stylesheet, an embedded stylesheet, or inline styles
  • CSS plays a crucial role in the design and presentation of modern web pages

A Simple HTML Document

Example

h1 {
    color:red
    background-color:red;
    padding:5px;
}
p {
    color:green
    background-color:yellow;
    padding:25px;
}
Try it Yourself »

Example Explained

  • The h1 selector applies the styles to all 'h1' elements on the page
  • The p selector applies the styles to all 'p' elements on the page
  • The color rule sets the text color of the selected elements.
  • The background-color rule sets the background color of the selected elements.
  • The padding rule sets the padding of of the selected elements. Padding is the space between the content of an element and its border.

CSS Saves a Lot of Work!

The style definitions are normally saved in external .css files.
With an external stylesheet file, you can change the look of an entire website by changing just one file!

Note: Before starting CSS tutorials, please make sure you are proficient in HTML.

CSS Syntax

❮ Previous Next ❯

A CSS rule set contains a selector and a declaration block.

HTML Documents

Selector: Selector indicates the HTML element you want to style. It could be any tag like h1, title etc.

Declaration Block: The declaration block can contain one or more declarations separated by a semicolon.

Property: A Property is a type of attribute of HTML element. It could be color, border etc.

Values: Values are assigned to CSS properties.

Example

p {
     color : green
     background-color : yellow;
     padding : 25px;
}
Try it Yourself

Example Explained

p is a selector in CSS it points to the HTML element you want to style.

color is a property, and green is the value of the property.

background-color is a property, and yellow is the value of the property.

padding is a property, and 25px is the value of the property.

Note: Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

CSS Colors

❮ Previous Next ❯

The color property in CSS is used to set the color of HTML elements.

In CSS, we use color values for specifying the color. We can also use this property for the border-color and other decorative effects.
We can define the color of an element by using the following ways:

  • RGB format
  • RGBA format
  • Hexadecimal notation
  • HSL
  • HSLA
  • Built-in color names

Let's understand the syntax and description of the above ways in detail.

RGB Format

RGB format is the short form of 'RED , GREEN and BLUE' that is used for defining the color of an HTML element simply by specifying the values of R , G and B that are in the range of 0 to 255.

Example

#rgb {
       color : rgb(255,0,0);
}
Try it Yourself »

RGBA Format

It is almost similar to RGB format except that RGBA contains A (Alpha) that specifies the element's transparency.
The value of alpha is in the range 0.0 to 1.0, in which 0.0 is for fully transparent, and 1.0 is for not transparent.

Example

#rgba {
       color : rgb(255,0,0,0.6);
}
Try it Yourself »

Hexadecimal Notation

Hexadecimal can be defined as a six-digit color representation.
This notation starts with the # symbol followed by six characters ranges from 0 to F.
In hexadecimal notation, the first two digits represent the red (RR) color value,
the next two digits represent the green (GG) color value, and the last two digits represent the blue (BB) color value.

Example

#hexadecimal {
       color : #(0-F)(0-F)(0-F)(0-F)(0-F)(0-F);
}
Try it Yourself »

HSL Notation

It is a short form of Hue, Saturation, and Lightness.

  • Hue: It can be defined as the degree on the color wheel from 0 to 360.
    0 represents red, 120 represents green, 240 represents blue.
  • Saturation: It takes value in percentage in which 100% represents fully saturated,
  • Lightness: The lightness of the color can be defined as the light that we want to provide the color

Example

#hsl {
       color : hsl(H,S,L);
}
Try it Yourself »

HSLA Notation

It is entirely similar to HSL property, except that it contains A (alpha) that specifies the element's transparency.
The value of alpha is in the range 0.0 to 1.0, in which 0.0 indicates fully transparent, and 1.0 indicates not transparent.

Example

#hsla {
       color : hsla(H, S, L, A);
}
Try it Yourself »

CSS Border

❮ Previous Next ❯

The CSS border is a shorthand property used to set the border on an element.

Syntax

The CSS border properties are use to specify the style, color and size of the border of an element.
The CSS border properties are given below :

  • border-style
  • border-color
  • border-width
  • border-radius

CSS border-style

The border-style property is used to specify the border type which you want to display on the web page.

The following values are used :

  • none : It doesn't define any border.
  • dotted : It is used to define a dotted border.
  • dashed : It is used to define a dashed border.
  • solid : It is used to define a solid border.
  • double : It defines two borders wIth the same border-width value.
  • groove : It defines a 3d grooved border. effect is generated according to border-color value.
  • ridge : It defines a 3d ridged border. effect is generated according to border-color value.
  • inset : It defines a 3d inset border. effect is generated according to border-color value.
  • outset : It defines a 3d outset border. effect is generated according to border-color value.

Example

p.none {   border-style : none   }
p.dotted {   border-style : dotted   }
p.dashed {   border-style : dashed   }
p.solid {   border-style : solid   }
p.double {   border-style : double   }
p.groove {   border-style : groove   }
p.ridge {   border-style : ridge   }
p.inset {   border-style : inset   }
p.outset {   border-style : outset   }

Try it Yourself

CSS border-width

The border-width property is used to set the border's width. It is usually set in pixels.

You can also use the one of the three pre-defined values, thin, medium or thick to set the width of the border.

Example

p.one {   border-style : solid; border-width : 6px;   }
p.next {   border-style : dotted; border-width : slim;   }
p.after {   border-style : dashed; border-width : medium;   }
p.last {   border-style : solid; border-width : thick;   }

Result

solid 6px

dotted slim

dashed medium

solid thick

Try it Yourself

Note: The border-color property is not used alone. It is always used with other border properties like "border-style" property to set the border first otherwise it will not work.

CSS border-radius

The border-radius property sets the rounded borders and provides the rounded corners around an element, tags, or div.
It defines the radius of the corners of an element.

It is shorthand for border-top-left-radius, border-top-right-radius, border-bottom-right-radius and border-bottom-left-radius.

Example

p.last {   border-style : solid; border-width : thick; border-radius : 10px;   }

Result

solid thick rounded
